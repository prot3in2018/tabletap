TableTap is a web-based restaurant ordering system that allows:
1. staff to manage digital menus
2. staff to generate QR code
3. staff to manage order
4. customers to place orders by scanning QR codes at their table

Setup Instructions:
1. Clone or download the repository.
2. Navigate to the project folder: cd tabletap
3. Run the server: python manage.py runserver
4. Visit http://localhost:8000 in your browser.

High-level view of each page:
- Dashboard: Central hub linking to all main features.
- Digital Menu: Staff can add, view, and delete menu items grouped by category.
- Table Setup: Staff can generate and download unique QR codes for each table.
- Order Management: Staff can view, complete,cancel, or clear orders
- Customer Ordering: Customers scan a QR code and place orders from the digital menu.
- Order History: Customers can view their own past orders.
- Account Management: Users can sign up, log in, and delete their accounts.
